# SolarXanh — Website bán thiết bị điện mặt trời

Website tĩnh (HTML/CSS/JS thuần, không cần build) cho SolarXanh — đơn vị lắp đặt và bán lẻ thiết bị điện mặt trời tại Việt Nam.

🌐 **Website chính thức:** [solarxanh.net](https://solarxanh.net)

## Cấu trúc dự án

```
├── index.html          # Trang chủ (gói giải pháp, ước tính chi phí, quy trình...)
├── san-pham.html        # Danh mục thiết bị bán lẻ (tấm pin, pin lưu trữ, inverter)
├── ve-chung-toi.html    # Giới thiệu công ty, đội ngũ, hành trình phát triển
└── lien-he.html         # Thông tin liên hệ, form đăng ký khảo sát, FAQ
```

## Công nghệ sử dụng

- HTML/CSS/JavaScript thuần — không cần cài đặt, không cần build
- Font: Be Vietnam Pro, Inter, JetBrains Mono (Google Fonts)
- Hosting: [Netlify](https://netlify.com)
- Tên miền: quản lý qua iNET

## Cách chỉnh sửa nội dung

Mỗi file `.html` là một trang độc lập, có thể mở bằng bất kỳ trình soạn thảo nào (kể cả trình soạn thảo trực tiếp trên GitHub — bấm biểu tượng ✏️ ở góc file) để sửa văn bản, giá cả, thông tin liên hệ...

Sau khi sửa và lưu (commit) trên GitHub, nếu đã kết nối với Netlify, website sẽ **tự động cập nhật** trong vòng 1-2 phút — không cần thao tác gì thêm.
